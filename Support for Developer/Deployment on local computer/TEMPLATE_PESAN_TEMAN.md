## 🎯 Tips Menulis Pesan

### DO ✅
- Sertakan URL yang jelas
- Berikan kredensial login
- Jelaskan fitur utama
- Minta feedback
- Sebutkan tech stack (untuk developer)
- Kasih warning laptop harus nyala

### DON'T ❌
- Jangan lupa ganti [PASTE_URL_TUNNEL_DISINI]
- Jangan share URL tanpa kredensial
- Jangan lupa mention ini masih development
- Jangan lupa minta feedback

## 📝 Checklist Sebelum Kirim

- [ ] URL tunnel sudah dicopy
- [ ] URL sudah diganti di template
- [ ] Akun demo sudah dibuat
- [ ] Laptop terkoneksi internet
- [ ] Laragon sudah running
- [ ] Sudah test akses sendiri
- [ ] Database ada data sample

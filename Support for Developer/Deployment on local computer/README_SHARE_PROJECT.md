# 📚 Panduan Share Project ke Teman

## 📁 File-File yang Tersedia

### 🚀 Script Siap Pakai
1. **`share-localtunnel.bat`** - Share via LocalTunnel (paling mudah, no signup)
2. **`share-ngrok.bat`** - Share via Ngrok (paling stabil)
3. **`setup-demo-account.bat`** - Buat akun demo untuk teman

### 📖 Dokumentasi
1. **`QUICK_START_SHARE.md`** - Panduan cepat 5 menit ⭐ **BACA INI DULU**
2. **`SHARE_PROJECT_GRATIS.md`** - Panduan lengkap semua metode

---

## ⚡ Quick Start (Pilih Salah Satu)

### Opsi 1: LocalTunnel (Termudah)
```bash
# 1. Install (sekali aja)
npm install -g localtunnel

# 2. Jalankan
Double-click: share-localtunnel.bat

# 3. Copy URL yang muncul, kirim ke teman
```

### Opsi 2: Ngrok (Terstabil)
```bash
# 1. Download & setup (sekali aja)
- Download: https://ngrok.com/download
- Daftar gratis: https://dashboard.ngrok.com/signup
- Setup: ngrok config add-authtoken YOUR_TOKEN

# 2. Jalankan
Double-click: share-ngrok.bat

# 3. Copy URL yang muncul, kirim ke teman
```
